1.技术构建：Vue3 + Vite + Pinia + Axios。
2.不再局限于 HBuilder X 内使用，任何编辑器都可以。
3.安装依赖/插件、运行命令等操作依照vue项目进行。
4.编写方式继续使用uniapp开发语言进行编码（可继续生成多端应用）。


# 克隆项目
git clone https://www.gitgl.com/R-and-D-center/soft-project/base/web/uniapp-v3.git
# 安装依赖
pnpm install
